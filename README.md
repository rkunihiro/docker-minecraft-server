# Minecraft(Java Edition) Server on Docker

## Running Spigot build tool
https://www.spigotmc.org/wiki/buildtools/#running-buildtools
mv spigot-1.x.x.jar spigot-server.jar

## Start server
```
$ cd /path/to/this/project
$ docker-compose up -d
```
